Program pozwala na zaawansowane obliczenia macierzowe z wykorzystaniem liczb zespolonych. Główne funkcjonalności to:

Wczytywanie macierzy zespolonych w formacie tekstowym (np. 3+2i, -4i).

Obliczanie iloczynu Frobeniusa (uwzględniającego sprzężenie zespolone).

Wyznaczanie normy Frobeniusa macierzy.

Obliczanie odległości oraz kąta między macierzami w sensie geometrycznym.
Projekt zawiera kod źródłowy oraz dokumentację PDF wyjaśniającą matematyczne podstawy operacji.
